## Zp Boilerplate

The purpose of this boilerplate is to start new projects using a React stack.

An immediate target is my home page.

## Stack

### Current

- ReactJs

- NextJs 

- NodeJs

- ExpressJs

- SocketIo

- SemanticUI-React

### TODO

- Redux

- Firebase

- Mongo

- Docker

- React Native

- Testing

## Instructions

```shell
git clone ...
cd ...
npm install
npm start
```
